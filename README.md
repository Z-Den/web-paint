# Web Paint App

## Features
### MVP
* draw and erase
* custom brush / eraser size
* custom brush color

![MVP](https://github.com/user-attachments/assets/50fbf4d4-fe51-48f0-a222-03f6b3dc1aec)

### Additional
* multiple canvases in tabs
* add, delete and rename tab function

![Additional](https://github.com/user-attachments/assets/ac039f05-5489-4f79-87dd-a9b8727f7abc)

### Experimental
* wipe canvas
* save image
* set drawing as page background

![Experimental](https://github.com/user-attachments/assets/ba0a8216-d3fe-4448-a8ef-5b611e37a61c)
